# GFT Star 3 - Teste Tecnico
Este repositório contém a resolução dos exercicios solicitados na fase tecnica do GFT Start 3.
## Informações do repositório
* A versão do Java utilizado foi Java 11 junto com a IDE Intellij
* Na pasta src está presente a resolução dos exercicios, sendo que os memos estão separados por pastas que contém a nomeclatura "Exercicio" seguido do seu numero.
